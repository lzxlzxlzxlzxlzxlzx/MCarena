# 我的世界怪物竞技场项目

## 项目概述
本项目是一个基于"我的世界"游戏的怪物竞技场模拟系统，包含测试主程序、自动测试用例生成工具、测试结果和怪物数据图鉴，旨在提供怪物战斗模拟与测试的完整解决方案。

## 文件说明

### 核心程序文件
- **importtime.py**：测试主程序，负责执行怪物竞技场的测试流程
- **importos.py**：测试用例生成工具，通过调用DeepSeek大模型自动生成测试用例

### 测试相关文件
- **output.csv**：自动生成的测试用例数据文件
- **批量测试结果.md**：测试执行结果的详细报告文档

### 数据文件
- **怪物图鉴.xlsx**：怪物属性数据表格（原始数据）
- **怪物图鉴.md**：怪物属性数据的Markdown格式文档（解析后版本）

## 使用方法

### 生成测试用例
```python
python importos.py
```
执行后将自动生成测试用例并保存至output.csv文件

### 运行测试程序
```python
python importtime.py
```
将读取output.csv中的测试用例，执行测试并生成测试结果

## 测试结果说明
测试结果详细记录于**批量测试结果.md**文件中，包含各测试用例的执行情况、怪物表现数据及分析结论。

## 怪物数据说明
怪物属性数据存储于**怪物图鉴.xlsx**及解析后的**怪物图鉴.md**中，包含怪物名称、生命值、攻击力、防御力等关键属性信息，为竞技场模拟提供基础数据支持。
